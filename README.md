UnicodeInsert
=============

Sublime Text 2 package to insert Unicode encoded special characters by either Unicode name or LaTeX command.

Default Hotkeys
---------------

- Mac OS X: **cmd+shift+c**
- Windows: **alt+shift+c**
- Linux: **super+shift+c**
